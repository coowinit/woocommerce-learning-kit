学习说明

这版不重点讲 PHP、hooks 和复杂二次开发，而是回到页面本身：先学会如何用 HTML + CSS 做出 WooCommerce 常见页面。

学完后，你会更容易理解 WooCommerce 的产品列表模板、产品详情模板、购物车模板和结账模板。
